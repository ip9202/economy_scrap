# RateStance 제품 개요

## 프로젝트 정보

- **프로젝트 명**: RateStance
- **프로젝트 타입**: Python 백엔드 + Next.js 대시보드
- **생성 일자**: 2026-01-27
- **상태**: 운영 중
- **버전**: 0.1.0

## 프로젝트 목적

통화정책 뉴스 스탠스 분석 파이프라인 및 대시보드. 한국은행(BoK) 기준금리 결정과 금융 뉴스 감성(매파/비둘기파) 간의 관계를 자동으로 분석하는 데이터 파이프라인입니다.

## 해결하는 문제

| 문제 | 설명 |
|------|------|
| **정책 신호 포착 지연** | 기자·분석가가 수동으로 뉴스를 검토해야 해 시간이 많이 소요 |
| **히스토리 데이터 부재** | 과거 금리 결정 전후 여론 동향 파악 어려움 |
| **정량화 부재** | 뉴스 논조를 수치로 비교·추적하기 어려움 |

## 핵심 기능

### 이중 소스 뉴스 수집

- **과거 데이터 (2025-08-01 이전)**: GDELT BigQuery 공개 데이터셋 활용
- **현재 데이터 (2025-08-01 이후)**: Google News RSS 피드 파싱
- **검색 키워드**: "한국은행 기준금리", "통화정책", "금리" 등

### 스탠스 스코어링

키워드 기반 감성 분석:
- **매파 (Hawkish)**: 인상, 긴축, 물가압력, 과열 등
- **비둘기파 (Dovish)**: 인하, 완화, 둔화, 부양 등
- **점수**: 양수(매파) / 음수(비둘기파) / 0(중립)

### 데이터 파이프라인

6단계 순차 처리:
1. 뉴스 수집 (GDELT/RSS)
2. 스탠스 스코어링 (키워드 매칭)
3. 일별 집계 (날짜별 평균 스탠스)
4. 한국은행 기준금리 수집 (ECOS API)
5. 이벤트 감지 (금리 변동 분류)
6. 이벤트 스터디 (변동 전후 ±14일 윈도우 분석)

### 대시보드 시각화

- 통계 카드 (총 기사, 평균 스탠스, 이벤트, 최근 이벤트)
- 뉴스 스탠스 시계열 차트 (클릭 가능 - 기사 드릴다운)
- 기준금리 시계열 차트
- 이벤트 스터디 차트 (유형별 ±14일 프로파일)
- 비동기 스크래핑 진행률 다이얼로그

### REST API

**데이터 조회:**
- `GET /api/data/news-daily`: 일별 뉴스 스탠스 집계
- `GET /api/data/rate-series`: 기준금리 시계열
- `GET /api/data/events`: 감지된 금리 변동 이벤트
- `GET /api/data/event-study`: 이벤트 스터디 분석 결과
- `GET /api/data/news-articles`: 개별 뉴스 기사 (페이지네이션)

**데이터 갱신 (비동기):**
- `POST /api/data/refresh`: 파이프라인 실행 (Job 생성)
- `GET /api/data/refresh/status/{job_id}`: Job 상태 조회
- `GET /api/data/refresh/jobs`: 전체 Job 목록
